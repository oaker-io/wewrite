# mockup-macos-app · macOS 应用窗口截图风

干净、专业的桌面 app 窗口截图美学:顶部交通灯按钮 + 工具栏 + 圆角 + 毛玻璃。适合让读者一眼认出"这是一个真实的 macOS 工具"。

## 视觉关键词

- Top-left traffic-light buttons (red/yellow/green) — 三个 12px 圆点 · 紧贴左上角
- Slim toolbar with SF Symbols icons · 垂直居中 · 间距均匀
- Rounded window corners (radius ~10px) · 整体大圆角
- Subtle drop shadow · 模拟悬浮窗效果
- Frosted-glass / vibrancy translucent background · 微妙背景模糊
- Sidebar (optional · 左侧 200px 浅灰栏) + main content area
- SF Pro Display / SF Pro Text 字体 · macOS 系统字
- 1x retina-sharp pixels · 不要任何手绘感
- 内容区可放代码块 / chart / 文本 — 取决于教程主题

## 适用场景

- AI 工具评测(Cursor、Raycast、Arc、Linear 等 macOS 原生 app)
- 桌面 app 教程 / 配置截图描述
- IDE / Cursor / VSCode 类(配 mockup-code-editor 也行)
- T2 工具评测型封面 + chart-1 截图

## 配色建议

- 浅色模式:窗口背景 #FFFFFF / 工具栏 #F5F5F7 / 文字 #1D1D1F / 强调色 #007AFF
- 深色模式:窗口背景 #1E1E1E / 工具栏 #2C2C2E / 文字 #F5F5F7 / 强调色 #0A84FF
- traffic-light 固定:红 #FF5F57 · 黄 #FEBC2E · 绿 #28C840

## 避免

- 不要 Windows / Linux 风格(没 traffic-light 就不像)
- 不要拟物化阴影或玻璃质感(macOS 现代风是扁平 + 微毛玻璃)
- 不要把 traffic-light 放右边(那是 Windows)
- 不要画太多 UI 装饰元素 — macOS 的精髓是"克制留白"
